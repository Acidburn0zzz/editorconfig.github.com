const char* editorconfig_parsing_info::conf_file_name

The file name of EditorConfig conf file. If this pointer is set to NULL, the file name is set to ".editorconfig" by default.

char* editorconfig_parsing_info::err_file

When a parsing error occured, this will point to a file that caused the parsing error.
